Module Tjr_fs_shared__Blk_intf

Blk-related interfaces

The main choices:

  • blk implementation (string, bytes, buf/bigarray etc)
  • blk_id (presumably int, but perhaps a pair of (dev,int) )
  • blk_dev (in mem, on file, on dev etc)
